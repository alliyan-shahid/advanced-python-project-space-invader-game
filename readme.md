Space Invaders - Ultimate Edition
A modern, feature-rich Space Invaders game built with Pygame. This project features stunning visual effects, power-ups, dynamic gameplay, and a modular codebase.

Project Structure
The game is organized into 10 modular files:

main.py - Entry point and game initialization

game.py - Main game loop and state management

player.py - Player ship logic and controls

enemy.py - Enemy behavior and AI

bullet.py - Projectile system for both player and enemies

particle.py - Particle effects for explosions and visual polish

powerup.py - Power-up system with various enhancements

star.py - Background starfield animation

death_explosion.py - Spectacular death animations

countdown.py - Wave start countdown sequences

🎮 Features
Core Gameplay
Classic Space Invaders mechanics with modern twists

Multiple enemy types (fighters, bombers, scouts)

Progressive difficulty with increasing waves

Score multiplier system

Lives system with visual feedback

Power-Up System
Double Shot: Permanent bullet upgrade

Shield: Temporary invincibility

Rapid Fire: Increased firing rate

Extra Life: Gain additional lives (max 5)

Score Boost: Double points for limited time

Visual Effects
Particle explosion systems

Animated starfield background

Dynamic bullet trails and glow effects

Screen shake and hit feedback

Wave transition animations

Countdown sequences

Power-up visual indicators

Controls
Arrow Keys: Move player ship

Spacebar: Shoot

R: Restart game

ESC: Quit game
